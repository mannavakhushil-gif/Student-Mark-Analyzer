# Student Marks Analyzer

A Python project to analyze student marks, calculate averages, find toppers, assign grades, and visualize performance using Matplotlib.

## Features
- Load student marks from a CSV file
- Calculate subject-wise averages
- Identify toppers for each subject
- Assign grades based on percentage
- Visualize subject-wise performance using bar charts

## Technologies
- Python
- Pandas
- Matplotlib

## How to Run
1. Make sure you have Python installed (version 3.x recommended)
2. Install required libraries if not already installed:
   ```bash
   pip install pandas matplotlib
